### Deploy AWS

Projeto utilizado para exemplo de deploy utilizando a AWS

## Como baixar o projeto?

```bash
    // Clonar repositório
    $ git clone https://github.com/danileao/deploy-aws-example

    // Acessar diretório
    $ cd deploy-aws-example

    // Instalar dependências
    $ npm install

    // Iniciar projeto
    $ npm run dev
```
